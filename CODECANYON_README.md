## 📂 Project Structure
- `documentation/` — Contains user manuals and setup guides. (MUST BE IN THE ZIP)
- `node_modules/` — (DO NOT INCLUDE IN ZIP)
- `style.css` — High-end styling with CSS variables.
- `app.js` — Core logic including CSV export and SQLite management.
- `main.js` — Electron entry point.
- `package.json` — Build configuration for .exe and .dmg.

## 🚀 How to prepare your Final ZIP for CodeCanyon
1.  Open `package.json` and replace `"Your Name"` with your actual CodeCanyon username.
2.  Run `npm install` followed by `npm run build` to ensure the app compiles.
3.  Create a folder named `FreelanceTaskManager_Main_File`.
4.  Inside it, put:
    - `Source Code` (All files except `node_modules` and `dist`)
    - `Documentation` (The `documentation` folder)
    - `Licensing` (Place the license files here)
5.  ZIP the `FreelanceTaskManager_Main_File` and upload to CodeCanyon.
